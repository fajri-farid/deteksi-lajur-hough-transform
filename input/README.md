# Folder Input Video

Folder ini berisi file video input untuk diproses oleh program deteksi jalur jalan.

## File Video

| File | Keterangan |
| --- | --- |
| `video.mp4` | Video input untuk deteksi jalur jalan. |

## Link Download

Karena file video tidak disertakan dalam repository (ukuran file besar dan masuk `.gitignore`), Anda dapat mengunduhnya melalui Google Drive:

- [Google Drive - video.mp4](https://drive.google.com/file/d/1mJj8Vhi2tczT316UMwXE_9T9-mGWTJaL/view?usp=sharing)

## Cara Menggunakan

1. Download file `video.mp4` dari link di atas.
2. Letakkan file tersebut di folder `input/` ini.
3. Pastikan nama file adalah `video.mp4`.
4. Jalankan program dari folder versi1 atau versi2.
